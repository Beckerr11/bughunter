# BugHunter

Triagem inteligente de bugs para times de produto.

## Problema
Bugs chegam com contexto fraco, duplicidade e prioridade incerta.

## Publico
Times de produto, QA e suporte tecnico.

## MVP
- Cadastro padronizado com templates por categoria.
- Prioridade sugerida por impacto e urgencia.
- Deteccao de potenciais duplicidades.
- Acompanhamento de SLA por responsavel.

## Stack
- React + TypeScript + Vite
- Node + Express + TypeScript
- CI com lint/test/build/audit

## Setup
1. npm install --include=dev
2. npm run bootstrap
3. npm run dev

## Monetizacao
Licenca por workspace com pacote por volume.
